# Hospital Management System

## Project Description

A MySQL-based Hospital Management System that manages:

* Departments
* Doctors
* Patients

## Concepts Used

* Primary Key
* Foreign Key
* INNER JOIN
* Multi-Table JOIN
* Views

## Tables

1. department
2. doctors
3. patient
